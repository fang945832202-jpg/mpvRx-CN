# MpvRx-CN

## 关于本项目

此项目是基于原项目 **[Riteshp2001/mpvRx](https://github.com/Riteshp2001/mpvRx)** 的 **v1.5.0-preview.2** 版本汉化而来，并已同步原作者截至以下 Hash 提交的所有更新：

> `7a800deef92dd9836930c7008e5a9fa3158405f8`

在此 Hash 之后，原作者可能还有新的提交，本版本尚未纳入。待原项目趋于稳定后，本汉化版本将考虑把此 Hash 之后的所有新增提交再次融合进来，并发布新的 Release 版本。

---

## 修改声明

相比原项目，本汉化版本做出了以下修改：

1. 移除了设置界面的赞助/捐赠板块
2. 移除了自动检查更新功能，仅保留手动检查更新按钮
3. 将 GitHub 仓库链接指向本仓库，以便用户获取汉化版本的后续更新
4. 完成了应用界面的全面中文本地化

## 说明

本人精力有限，难免有未汉化完全的部分。如果在后续使用中发现遗漏，欢迎在 GitHub 上提交 Issue，我会及时跟进更新。

另外需要特别说明的是，本次汉化工作很大程度上得益于 **[Riteshp2001](https://github.com/Riteshp2001)** 奠定的基础。原作者此前已经提供了一份中文版本的字符串文件，这极大地节省了工作量。我在此基础上，主要将一些硬编码（Hardcoded）文本进行了中文化。在此向原作者表示衷心的感谢。

如果各位在使用中文版的过程中遇到软件使用上的问题，建议直接去原项目地址提交 Issue，而不是在这里提出。因为本汉化版本并未修改任何功能层面的代码，仅修改了显示层面的文本。

如果发现某些功能在原版中可以正常使用，但在汉化版中无法使用，这大概率是汉化导致的问题。遇到这种情况，欢迎在 GitHub 上提交 Issue 反馈，我会及时跟进并解决。
