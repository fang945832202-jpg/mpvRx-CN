# MpvRx-CN

## 📖 关于本项目

此项目是基于 **[Riteshp2001/mpvRx](https://github.com/Riteshp2001/mpvRx)** 汉化而来的中文本地化版本。

---

## 🚧 当前状态

**🀄 目前正在进行汉化工作中，敬请期待！**

---

> 💡 **小提示：** mpvRx 项目实际上是 [marlboro-advance/mpvEx](https://github.com/marlboro-advance/mpvEx) 的一个分支。如果你想使用 mpvEx 的汉化版本，可以前往：[azxcvn/mpvEx-CN](https://github.com/azxcvn/mpvEx-CN)
>
> ⚠️ **注意：** mpvEx 和 mpvRx 的中文汉化均为作者个人行为，与原官方项目无任何关系，也无任何关联。
